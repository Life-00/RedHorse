# 교대근무자 생체리듬 최적화 앱 - 완성본

## 🎉 완성된 기능

### 📱 3개 핵심 화면
1. **온보딩 화면** - 근무 형태 선택 (2교대, 3교대, 고정야간, 불규칙)
2. **웨어러블 연결** - Apple Health, Google Fit 연동 옵션
3. **홈 대시보드** - 개인화된 수면 계획 및 건강 지표

### 🎨 프리미엄 디자인
- **브랜드 컬러**: #5d5cff (메인), #ff8a00 (포인트)
- **모던 UI**: 32px 둥근 모서리, 그라데이션 카드
- **모바일 최적화**: 최대 너비 448px (max-w-md)
- **한국어 UI**: 교대근무자 맞춤 콘텐츠

### 💡 핵심 기능
- **Shift-to-Sleep 플랜**: 개인화된 수면 시간 추천
- **카페인 컷오프**: 최적 카페인 섭취 시간 가이드
- **실시간 건강 지표**: 수면효율, 피로도, 집중력, 심박수
- **오늘의 팁**: 맞춤형 건강 관리 조언

## 🛠 기술 스택

- **프론트엔드**: HTML5, CSS3, JavaScript
- **스타일링**: Tailwind CSS (CDN)
- **아이콘**: Lucide Icons
- **반응형**: 모바일 퍼스트 디자인

## 🎨 디자인 특징

### 색상 팔레트
- **Primary**: #5d5cff (보라)
- **Secondary**: #ff8a00 (오렌지)
- **Background**: #ffffff (화이트)
- **Gray**: #f9fafb (연한 회색)

### UI 컴포넌트
- **카드**: 32px 둥근 모서리
- **버튼**: 호버 효과, 스케일 애니메이션
- **그라데이션**: 부드러운 색상 전환
- **타이포그래피**: 시스템 폰트, 가독성 최적화

## 📊 화면별 상세 기능

### 1️⃣ 온보딩 화면
- 4가지 근무 형태 선택
- 진행률 표시 (점 인디케이터)
- 각 옵션별 아이콘과 설명

### 2️⃣ 웨어러블 연결
- Apple Health, Google Fit 연동 옵션
- 개인정보 보호 안내
- 건너뛰기 기능

### 3️⃣ 홈 대시보드
- **개인화 인사**: "안녕하세요, 지연님"
- **오늘 근무**: 야간 22:00-07:00 표시
- **수면 계획**: 08:00-15:30 최적 수면창
- **카페인 가이드**: 21:00 이전 섭취 권장
- **건강 지표**: 4개 카드로 현황 표시
- **맞춤 팁**: 3가지 실용적 조언

## 🎯 타겟 사용자
